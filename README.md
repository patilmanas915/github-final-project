# Simple Interest Calculator

This project is a Bash-based Simple Interest Calculator that helps users calculate interest based on principal, rate, and time.

## 📌 Features
- Easy-to-use command-line interface
- Takes user input dynamically
- Calculates simple interest instantly

## 🧮 Formula
Simple Interest = (Principal × Rate × Time) / 100

## 🚀 How to Run

1. Open terminal
2. Run the script:
   bash simple-interest.sh

3. Enter:
   - Principal amount
   - Rate of interest
   - Time period

## 📊 Example

Input:
Principal = 1000  
Rate = 5  
Time = 2  

Output:
Simple Interest = 100
